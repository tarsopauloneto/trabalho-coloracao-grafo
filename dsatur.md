# Trabalho — Coloração de Grafos

## DSatur

O DSatur é uma heurística para coloração de grafos.

### Ideia:

Escolher sempre o vértice com maior grau de saturação.

### Grau de saturação:

Número de cores diferentes nos vizinhos.

### Passos:

1. Escolher vértice de maior grau
2. Colorir com menor cor possível
3. Repetir escolhendo maior saturação
4. Desempate: maior grau

### Observação:

Não garante número mínimo de cores.
